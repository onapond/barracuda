
# Barracuda Coffee / Roastery Page — Codex PRD
Version: 1.0

이 문서는 Barracuda 웹사이트의 **Coffee / Roastery 페이지** 구현을 위한 개발 명세서이다.
Codex가 추가 질문 없이 페이지를 구현할 수 있도록 작성되었다.

--------------------------------------------------
1. 페이지 목적
--------------------------------------------------

Coffee 페이지는 다음 목적을 가진다.

1. Barracuda 커피의 신뢰 형성
2. CNC Tech 로스터리 소개
3. 로스팅 철학 전달
4. 원두 브랜드 이미지 형성
5. 향후 원두 판매 페이지 연결

Barracuda는 단순 카페가 아니라 **로스터리 기반 커피 브랜드**라는 인식을 만들어야 한다.
